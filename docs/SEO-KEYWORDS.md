## 0. Cómo se busca este nicho (intención del usuario)

Hay **3 idiomas de búsqueda** que conviven en el mismo nicho. El productor debe atacar los tres:

| Término | Idioma/Región | Quién lo usa |
|---|---|---|
| `type beat` | EN (estándar global, también usado en Latam) | Productores, artistas que buscan en YouTube |
| `instrumental` / `instrumentales` | ES | Artistas latinos, cantantes |
| `base` / `bases de rap` | ES (España + Latam) | Raperos, freestylers |
| `pista` / `pistas` | ES (Latam, reggaetón) | Cantantes urbanos |
| `beat` / `beats` | universal | Todos |

**Regla de oro del nicho:** la búsqueda real casi nunca es genérica. Es **`[artista] type beat`** o **`[subgénero] + [mood] + [año]`**. Las páginas que rankean combinan artista × subgénero × mood × BPM.

## 3. MODIFICADORES (multiplican cada keyword)

La fórmula ganadora es: **[Artista] x [Artista] Type Beat — "[Mood]" | [Año] [Subgénero] [BPM]**

### Modificadores de MOOD (estado de ánimo) — muy buscados
dark · melodic · sad · emotional · chill · aggressive · hard · smooth · ambient · dreamy · nostalgic
→ ES: oscuro · melódico · triste · emotivo · agresivo · duro · relajado

### Modificadores de FORMATO (en breakout)
- **sped up** (sped up type beat) ⬆️
- **slowed** / slowed + reverb
- **chopped**

### Modificadores TÉCNICOS (búsquedas de cola larga, altísima intención)
- BPM: `140 bpm trap type beat`, `trap type beat 140`
- Tonalidad/Key: `type beat C minor`, `instrumental en La menor`
- Año: `type beat 2025`, `type beat 2026`
- `free` vs de pago: `free type beat` (tráfico) vs sin "free" (compra)

### Combos ARTISTA × SUBGÉNERO (los que más rankean en YouTube)
Ejemplos del patrón ganador (adaptar a referentes del artista de Maxi):
- `drake type beat dark melodic`
- `travis scott type beat`
- `future x drake type beat`
- `afro rnb x brazilian funk type beat`
- Referentes urbano-latino: `feid type beat`, `bad bunny type beat`, `myke towers type beat`, `duki type beat`, `bzrp type beat`

> ⚠️ Evita tags genéricos como "music" o solo el nombre del artista — atraen oyentes casuales, no compradores. Prioriza **buyer-intent**.

## 9. 🤖 GEO — aparecer en ChatGPT, Perplexity, Google AI Overviews

> Gartner proyecta −25% de tráfico orgánico para 2026 por el cambio a buscadores con IA. Estar en las respuestas generativas ya no es opcional.

- **Permitir crawlers de IA** (hecho en `robots.ts`; falta confirmar en panel Cloudflare — ver `SEO-PLAN.md`).
- **Contenido server-rendered** (la IA no ejecuta JS bien) → FAQ y datos clave ya van en el HTML.
- **JSON-LD** (`MusicGroup`, `Product`, `FAQPage`) → da hechos estructurados que la IA cita.
- **Formato de respuesta directa**: bloques pregunta→respuesta cortos y claros (FAQ) = lo que la IA extrae.
- **Frescura**: marca de fecha / "actualizado 2026" en contenido de tendencias.
- **Métricas GEO**: *Mention Rate* (¿la IA nombra la marca?), *Citation Rate* (¿enlaza al dominio?), *Position*. Medir con prompts manuales en ChatGPT/Perplexity periódicamente.
